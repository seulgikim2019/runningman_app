# runningman_app
제한 시간 4분 동안, 프로그램 [런닝맨]처럼 캐릭터(이름표)나 점령지를 카메라로 촬영하여 승리하는 게임을 진행할 수 있는 어플. 



기능
----
1. 회원가입, 로그인
2. 게임방 생성, 게임방 입장
3. 카메라 촬영(학습된 이미지 : 점령지 4개, 캐릭터 4개)
4. 제한시간(4분) 후 우승자 결정



사용기술
--------
● 서버: AWS

● 웹 서버: Apache, Flask

● 데이터베이스: Mysql

● 언어: Java, Python, PHP

● 라이브러리/API : TensorFlow
